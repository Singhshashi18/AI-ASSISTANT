# AI Knowledge Assistant Frontend

React + Vite + Tailwind CSS + Redux Toolkit

## Setup

```bash
npm install
npm run dev
```

## Features

- User authentication (login/register)
- Dashboard with admin stats
- Chat interface
- Document management
- RAG search and ask capabilities
- Admin panel

## Structure

```
src/
├── components/     # Reusable components
├── pages/          # Page components
├── store/          # Redux store
├── services/       # API services
├── hooks/          # Custom hooks
├── utils/          # Utility functions
└── styles/         # Global styles
```
